🔧 SVG ID 编辑工具

此工具帮助您为 SVG 文件中的路径添加 ID 属性，使其在地图中可被点击和识别。
使用步骤： 1. 上传导出的 china-map.svg 文件 2. 系统会自动识别所有路径 3. 为每个路径添加对应的 ID（如 beijing, shanghai 等） 4. 点击"应用更改"保存 5. 点击"下载修改后的 SVG"获取新文件
