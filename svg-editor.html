🔧 SVG ID 编辑工具

此工具帮助您为 SVG 文件中的路径添加 ID 属性，使其在地图中可被点击和识别。
工作流程： 1. 上传 SVG 文件 → 系统自动从路径名称中提取 6 位数字作为 ID 2. 对于没有 6 位数字的路径，需要手动输入 ID 3. 检查所有 ID 是否正确 4. 点击"应用更改"和"下载修改后的 SVG"
