# Adobe Illustrator SVG 导出完整指南

## 📋 概述

本指南将帮助您将 `中国一级至三级行政区划地图.ai` 文件导出为可在地图应用中使用的 SVG 格式。

---

## 🎯 目标

1. ✅ 为每个地区的路径添加唯一的 ID
2. ✅ 导出为 SVG 格式
3. ✅ 保存为 `china-map.svg` 到项目文件夹

---

## 🔧 步骤 1: 打开 Illustrator 文件

### 1.1 启动 Adobe Illustrator
- 打开 Adobe Illustrator
- 文件 → 打开
- 选择 `中国一级至三级行政区划地图.ai`
- 等待文件完全加载（可能需要几秒钟）

### 1.2 检查文件结构
- 打开菜单：**Window → Layers** （或按 F2）
- 查看左侧图层面板
- 应该能看到各个省份的图层或组

## 🏷️ 步骤 2: 为地区添加 ID

### 方法 A：使用 SVG 属性面板（推荐）

这是最直接的方法。

#### 2A.1 选择第一个地区
- 在图层面板中选择第一个地区（如：北京）
- 或在画布上用选择工具点击该地区
- 应该看到地区被高亮选中

#### 2A.2 打开属性面板
- 右键点击选中的地区
- 选择 **"编辑内容"** 或 **"编辑"**
- 或在菜单栏：**Object → SVG Attributes** （取决于 Illustrator 版本）

#### 2A.3 添加 ID
- 在出现的属性或检查器面板中查找 **ID** 字段
- 输入地区的英文名称（如：`beijing`）
- 确保 ID 是：
  - ✓ 英文字母或数字
  - ✓ 不包含空格
  - ✓ 不包含特殊字符（除了下划线 _）
  - ✓ 全小写
- 按 Enter 确认

#### 2A.4 重复所有地区

## 📤 步骤 3: 清理文件

### 3.1 删除不必要的元素
在导出前，建议清理以下内容：

- [ ] 删除文字和标题
- [ ] 删除背景色或背景图片
- [ ] 删除装饰性元素

## 📁 步骤 4: 导出为 SVG

### 4.1 打开导出对话框

选择以下之一：
- **菜单**：文件 → 导出为... → 选择 SVG 格式
- **快捷键**：`Ctrl+Shift+E`（Windows）或 `Cmd+Shift+E`（Mac）

### 4.2 设置文件名和位置

在文件保存对话框中：

| 选项 | 设置 |
|------|------|
| **文件名** | `china-map` |
| **文件类型** | `SVG (*.svg)` |
| **位置** | 选择 `MapOfChina` 项目文件夹 |

- ✓ 确认路径正确
- ✓ 确认文件名为 `china-map`（不包括 .svg 扩展名）

### 4.3 点击 "导出" 按钮

此时会弹出 **"SVG 导出选项"** 对话框。

---

## ⚙️ 步骤 5: 配置 SVG 导出选项

这是最重要的一步！请按以下设置配置：

### 5.1 基本设置

```
SVG 版本：         ◉ SVG 1.1
类型：            ◉ 浮点型
小数位精度：        3
```

### 5.2 文本和字体设置

```
字体类型：         ◉ SVG
```

### 5.3 样式设置

**重要！** 这里的设置会影响地区的可交互性。

```
样式：            ◉ 内联样式 ✓ (必选！)
```

选择 "内联样式" 将 CSS 直接写入 SVG，确保样式被正确保存。

### 5.4 转换选项

```
☑️ 转换为路径          (推荐勾选)
☑️ 转换为文字轮廓      (根据需要)
☐ 响应式 SVG           (不勾选)
☐ 从 AI 保留            (不勾选)
```

**转换为路径** 很重要：
- ✓ 将所有文字和形状转换为路径
- ✓ 确保在任何软件中都能正确显示
- ✓ 使文件更加稳定

### 5.5 其他选项

```
十进制字数：  保持默认
保留 Illustrator 编辑能力：  ☐ (不勾选)
使用 <textPath> for 沿路径文本：  ☐ (根据需要)
输出渐变为：  ◉ <linearGradient> / <radialGradient>
保留裁切区域：  ☑️ (如需要)
```

### 5.6 总结导出设置

```
✓ SVG 版本：SVG 1.1
✓ 样式：内联样式
✓ 转换为路径：勾选
✓ 响应式 SVG：不勾选
✓ 从 AI 保留：不勾选
```

## ⚠️ 常见问题

### Q: 如何找到 SVG 属性面板？
**A:** 在较新版本的 Illustrator 中：
- 右键选中对象 → "编辑内容"
- 或菜单 → Object → SVG Attributes
- 或检查右侧的"属性"或"检查器"面板

如果找不到，可以使用 **svg-editor.html** 工具在导出后添加 ID。

### Q: 导出后 SVG 非常大？
**A:** 这是正常的。可以使用在线 SVG 优化工具：
- https://jakearchibald.github.io/svgomg/
- 上传您的 SVG 文件
- 下载优化后的版本

### Q: 导出的 SVG 中没有地区显示？
**A:** 可能原因：
1. 在导出选项中取消勾选了 "转换为路径"
2. 图层被隐藏或锁定
3. 地区使用了特殊的填充效果（如网格、图案）

解决方案：
- 重新导出，确保勾选 "转换为路径"
- 检查所有图层是否可见

### Q: ID 应该是什么？
**A:** ID 应该是：
- ✓ 英文字母（小写）
- ✓ 例如：beijing, shanghai, henan
- ✓ 不能有空格或特殊字符
- ✓ 对应的地区名称

### Q: 可以跳过添加 ID 吗？
**A:** 技术上可以，但不推荐。如果没有 ID：
- 地图无法正确交互
- 无法识别点击的地区
- 建议使用 svg-editor.html 添加 ID
