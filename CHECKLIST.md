## 🎯 常见问题快速检查

### 地图不显示？
- [ ] F12 打开开发者工具
- [ ] 查看 Console 标签是否有红色错误
- [ ] 查看 Network 标签，`china-map.svg` 状态是 200 还是 404
- [ ] 检查文件夹中是否有 `china-map.svg` 文件
- [ ] 确认文件名完全匹配（包括大小写）

### 点击地区没有反应？
- [ ] F12 Console 检查是否有错误信息
- [ ] 检查 `china-map.svg` 中的路径是否有 `id` 属性
  - 用文本编辑器打开 SVG
  - 搜索 `id=`
  - 应该能找到地区名称的 ID
- [ ] 使用 `svg-editor.html` 添加缺失的 ID

### 页面显示空白或加载慢？
- [ ] 检查浏览器控制台是否有错误
- [ ] 尝试清除浏览器缓存并重新加载（Ctrl+Shift+Delete）
- [ ] 如果使用本地文件打开，尝试用本地服务器运行
  ```bash
  python -m http.server 8000
  # 访问 http://localhost:8000
  ```

### 数据丢失？
- [ ] 检查是否清除了浏览器缓存
- [ ] 查看是否有之前导出的 JSON 备份
- [ ] 尝试从备份导入数据
- [ ] 记住定期导出数据！
